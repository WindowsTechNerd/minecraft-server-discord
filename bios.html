BIOS Setup

BIOS Setup

Configure your ThinkPad X1C4's firmware for optimal compatibility with OpenCore and macOS Catalina. These settings ensure UEFI booting, disable incompatible features, and prepare the system for Hackintosh installation.

Accessing BIOS

  1. Power on or restart your ThinkPad X1C4.
  2. Immediately press F1 repeatedly when the Lenovo logo appears to enter BIOS Setup.
  3. If you miss it, restart and try again.

Recommended BIOS Settings

Navigate through the BIOS menus and adjust the following settings:

Security Tab

  • Secure Boot: Disabled
  • Intel SGX: Disabled
  • Intel Platform Trust Technology: Disabled

Startup Tab

  • UEFI/Legacy Boot: UEFI Only
  • CSM Support: Disabled
  • Boot Mode: Quick

Restart Tab

  • OS Optimized Defaults: Disabled

Config Tab → Power

  • Sleep State: Linux (for better compatibility)

Config Tab → USB

  • Always On USB: Enabled

Config Tab → Keyboard/Mouse

  • Fn and Ctrl Key swap: Disabled
  • F1-F12 as Primary Function: Disabled

Saving Changes

  1. Press F10 to save changes and exit.
  2. Confirm by pressing Y when prompted.
  3. The system will restart with the new BIOS settings.

Troubleshooting

  • If you can't access BIOS, try pressing Enter then F1 during boot.
  • Ensure you're pressing the key before the Windows logo appears.
  • If settings don't stick, check for BIOS updates from Lenovo's website.
