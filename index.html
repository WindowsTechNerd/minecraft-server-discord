A complete, Catalina‑optimized EFI build guide for the ThinkPad X1C4. Stable Wi‑Fi, Bluetooth, audio, sleep, and full DRM support.

macOS Catalina OpenCore Intel Wi‑Fi Hackintosh

About This EFI

This guide teaches you how to build a clean, stable EFI for macOS Catalina (10.15) on the ThinkPad X1 Carbon 4th Gen. Catalina is the newest macOS version with full support for:
  • Intel Wi‑Fi (itlwm Catalina build)
  • Intel Bluetooth (Firmware + Injector)
  • Skylake iGPU acceleration
  • Safari DRM (Netflix, Apple TV+, Prime Video)

Hardware Profile

Supported configuration:
  • ThinkPad X1 Carbon 4th Gen
  • Intel Skylake CPU
  • Intel HD Graphics
  • Intel Dual Band Wireless‑AC 8260
  • Intel Bluetooth
  • Conexant CX11852 audio

EFI Build Guide (Catalina)

1. BIOS Setup

  • Disable Secure Boot
  • Enable UEFI only
  • Disable Legacy Boot
  • Set SATA to AHCI

2. Create Catalina Installer

sudo /Applications/Install\ macOS\ Catalina.app/Contents/Resources/createinstallmedia --volume /Volumes/USB

3. Add OpenCore Base Files

EFI/
  BOOT/BOOTx64.efi
  OC/
    OpenCore.efi
    ACPI/
    Kexts/
    Drivers/
    config.plist

4. Required Kexts

  • Lilu.kext
  • VirtualSMC.kext
  • WhateverGreen.kext
  • AppleALC.kext
  • VoodooPS2Controller.kext

5. Wi‑Fi (Catalina)

Use the Catalina build of:

itlwm.kext
Do NOT use AirportItlwm — that is for Big Sur and newer.

6. Bluetooth

IntelBluetoothFirmware.kext
IntelBluetoothInjector.kext

7. SMBIOS

MacBookPro13,1

Generate using GenSMBIOS:

  • Serial Number
  • MLB
  • System UUID
  • ROM (MAC address)

8. Install macOS

Boot from USB → Install Catalina → Copy EFI to internal drive.

Post‑Install Notes

Catalina provides the best stability for Skylake ThinkPads. DRM works, Intel Wi‑Fi is stable, and Bluetooth pairing is reliable.
